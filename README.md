# Movie Search Application

This is a simple web application that allows users to search for movies by title and view details such as the movie poster, year, genre, and plot. The application also provides suggestions based on user input.

## Features

- Search for movies using the OMDb API.
- Autocomplete suggestions as you type in the search bar (limited to 5 suggestions).
- Display movie details including the poster, title, year, genre, and plot.
- Responsive design for better user experience.

## Technologies Used

- HTML
- CSS
- JavaScript
- OMDb API (http://www.omdbapi.com/)
